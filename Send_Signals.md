- Port: 11608
- IP: 224.0.0.88

Format: "belled,\<router sn\>,\<router ip\>"

belled: recognize this signal is sent by MB8800 (Q Station)

router sn: sn number of Q Station

router IP: IP address of Q Station

```js
  {
      "cmd":"ping"
  }
  ```

  ```js
  belleds,<router sn>,<router ip>
```
